# Random Password Generator
Автор: Ермаков Кирилл
Описание: Программа для генерации надежных паролей с настройкой параметров. Создает простой пароль: Длина 8, только буквы "aBcDeFgH". Или сложный пароль: Длина 16, буквы, цифры и спецсимволы.
Примеры использования: 
1. Базовый (Пин-код)
Настройки: Длина 4, только «Цифры».
Результат: В истории появится запись вида 5829
2. Безопасный (Для соцсетей)
Настройки: Длина 12, «Буквы» + «Цифры».
Результат: Пароль формата x7K2pL9nQ1rW.
3. Максимальный (Для админа)
Настройки: Длина 32, все чекбоксы (Буквы + Цифры + Спецсимволы).
Результат: Сложная строка вроде j#8$K1!pL9*qR2@vN5^zX7&mB4(tY0_s.
